created-at.txt data processing¶

Calculates frequency data by:

  • account created_at
  • tweet id created_at

Also combines in diachronic HTTP status data (this data was collected in recurring phase samples, over several weeks. Each phase took a day or more. The first phase took a week, and really captured the fact that so many tweets and accounts have been removed from twitter. The question is by whom and why).

This notebook can be run using the Makefile target: make created-at (look at the makefile to see how the notebook is being run from the command line)
